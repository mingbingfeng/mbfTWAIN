# mbfTwain v1.0.3

这是 `v1.0.2` 的队列体验增强与交互修复版本。

## 新增内容

- UI 新增“记住最后选择的图片”复选框，默认不勾选。
- 勾选后会把用户上次选择的图片列表和旋转状态保存到本地用户配置。
- 后续每次打开 UI 或开始新的扫描会话时，会自动恢复上次保留的图片队列，省去重复选图。

## 修复内容

- 修复缩略图队列中操作按钮会在鼠标从缩略图本体移动到按钮时立刻消失的问题。
- 缩略图在 hover 和选中状态下都会稳定显示移动、旋转和删除操作，避免按钮闪烁或误触失败。
- 队列项 hover 状态会在移出整个卡片后再清除，保证缩略图操作的可点击性。

## 安装包

下载安装资产：

- `mbfTwain-Setup-v1.0.3.exe`
- `mbfTwain-Setup-v1.0.3.exe.sha256`

安装器需要管理员权限，因为 TWAIN source 需要写入 Windows 的 TWAIN 目录。
如果仓库保持私有，UI 更新检查需要在启动前设置可读取 release 的
`MBF_TWAIN_GITHUB_TOKEN`；公开仓库不需要令牌。
